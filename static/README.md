# Curso Gatsby
> Como criar um site PWA com React, GraphQL e Netlify CMS

O código deste repositório é baseado no Curso de Gatsby ministrado por Willian Justen.

## Como rodar o projeto localmente
```sl
# Baixar as dependências
yarn install

# Rodando o Gatsby
gatsby develop
```
